Parité

Définition: Soit \(f\) une fonction et soit \(D_{f}\) son ensemble de définition.
1. La fonction \(f\) est paire si et seulement si, pour tout \(x\) de \(D_{f}\), \(-x\) est élément de \(D_{f}\), et \(f(-x) = f(x)\). La courbe représentative de \(f\) dans un repère orthogonal admet alors l'axe des ordonnées comme l’axe de symétrie.
2. La fonction \(f\) est impaire si, et seulement si, pour tout \(x\) de \(D_{f}\), \(-x\) est élément de \(D_{f}\), et \(f(-x) = -f(x)\). La courbe représentative de \(f\) dans un repère cartésien admet alors l’origine comme centre de symétrie.

Explication: \(f(x)\) est paire si \(f(x) = f(-x)\). Dans ce cas l’axe d'équation \(x = 0\) est l’axe de symétrie. \(f(x)\) est impaire si \(f(-x) = -f(x)\). Dans ce cas l’origine \(0(0;0)\) est le centre de symétrie.

Diagrames:

Soit \(f(x) = 2x^2 + 5\)
\(f(-x) = 2(-x)^2 + 5 = 2x^2 + 5.\) Donc, \(f(x)= f(-x)\), alors \(f\) est un fonction paire. L’axe \(x = 0\), est l’axe centre de symétrie.


Soit \(f(x) = 2x^3 + 5x\)
\(f(-x) = 2(-x)^3 + 5(-x) = -2x^3 -5x.\) Donc \(f(-x) = -f(x)\), alors \(f\) est une fonction impaire. L’origine est le centre de symétrie.

Soit \(f(x) = 2x^2 + 5x + 1\)
\(f(-x) = 2x^2 - 5x + 1\) et \(-f(x) = -2x^2 - 5x -1.\) Donc comme \(f(x) \ne f(-x)\) et \(f(-x) \ne -f(x)\), \(f\) n’est ni paire ni impaire.

    Exemples :

  1. \(f(x) = 3x^2 - 5x - 2\)
    La parité: Cherchons nous pour la parité
    \(f(-x) = 3(-x)^2 - 5(-x)-2 = 3x^2 + 5x -2\)
    \(f(x) = -3x^2 - 5x + 2\)
    Donc comme \(f(x) \ne f(-x)\) et \(f(-x) \ne -f(x)\), \(f\) n’est ni paire ni impaire.

  2. \(f(x) = 10x^2 + 20\)
    Parité: Cherchons nous pour la parité
    \(f(-x) = 10x^2 + 20\)
    \(f(x) = f(-x)\)
    Donc, \(f\) est paire.

  3. \(f(x) = -13x^2 - x + 12\)
    Parité: Cherchons nous pour la parité
    \(f(-x) = -13x^2 + x + 12\)
    \(-f(x) = 13x^2 - x - 12\)
    Donc comme \(f(x) \ne f(-x)\) et \(f(-x) \ne - f(x)\), \(f\) n’est ni paire ni impaire.

  4. \(f(x) = \sqrt{x - 1}\)
    Parité:
    \(f(-x) = -x - 1\)
    \(-f(x) = -x - 1\)
    Donc, \(f\) n’est ni paire ni impaire.
    Une meilleur façon de voir: \(D_{f} = [1 ; ∞ [\)
    Pour tout \(x\) de \(D_{f} -x ∉ D_{f}\); donc ni paire ni impaire. En d’autre mots, le domaine n’est pas symétrique.

  5. \(f(x) = {8\over9x}\)
    Parité:
    \(f(-x) = {8\over(-9x)}\)
    \(-f(x) = {-8\over9x}\)
    Donc, \(f\) est impaire.

  6. \(f(x) = |x^2 - 3x + 2|\)
    Parité:
    \(f(-x) = |x^2 + 3x + 2|\)
    \(-f(x) = -|x^2 - 3x + 2|\)
    Donc comme \(f(x) \ne f(-x)\) et \(-f(x) \ne f(-x)\), \(f\) n’est ni paire ni impaire.

  7. \(f(x) = sin(x)\)
    \(f(-x) = sin(-x) = -sin(x)\)
    \(-f(x) = -sin(x)\)
    Donc comme \(-f(x) = f(-x)\), \(f\) est impaire.

  8. \(f(x) = cos(x)\)
    \(f(-x) = cos(-x) = cos(x)\)
    Donc comme \(f(x) = f(-x)\), \(f\) est paire.
